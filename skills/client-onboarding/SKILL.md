# Client Onboarding

Structured workflow for onboarding HENRY AI agency clients.

## Phase 1: Discovery (Week 1)
- [ ] Kickoff call — document current pain points
- [ ] Map existing workflows and tech stack
- [ ] Identify top 3 automation opportunities
- [ ] Calculate projected ROI for each opportunity
- [ ] Prioritize by impact/effort ratio

## Phase 2: Proposal & SOW
- [ ] Draft scope of work with phased delivery
- [ ] Price: $5K-$25K project based on complexity
- [ ] Include retainer upsell: $500-$2K/month for ongoing optimization
- [ ] Define success metrics and acceptance criteria
- [ ] Client sign-off

## Phase 3: Build (Weeks 2-4)
- [ ] Set up client environment (isolated, secure)
- [ ] Build automation solutions per SOW
- [ ] Follow superpowers TDD workflow
- [ ] Internal code review before client demo
- [ ] Client demo and feedback loop

## Phase 4: Deploy & Train
- [ ] Production deployment with monitoring
- [ ] Staff training sessions (record for async access)
- [ ] Documentation package delivered
- [ ] 30-day support period begins

## Phase 5: Retainer Transition
- [ ] 30-day review meeting
- [ ] Present optimization opportunities
- [ ] Transition to retainer agreement
- [ ] Set up monthly reporting cadence
- [ ] Log client in HENRY_CONTEXT.md

## Quality Gates
- No deployment without passing tests
- Client sign-off required at each phase
- All work tracked in Command Center
